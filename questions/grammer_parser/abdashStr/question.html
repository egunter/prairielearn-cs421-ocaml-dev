Problem

Consider the set of all strings over the alphabet $\{\ a,\; b,\;-,\; ; \; \}$ that represent non-empty sequences of semicolon separated dash-surrounded non-empty strings of $a$'s and $b$'s. A dash-surrounded non-empty string of $a$'s and $b$'s begins with a dash ($-$), followed by a non-zero length sequence of $a$'s and $b$'s, terminated by another dash. Such strings are separated by semicolons but there is no semicolon following the last such string.

For example, $-aababb-;-babaaba-;-a-$ follows the desired pattern; $--;-a-$ and $-aaba-;$ do not.

  • In the file abdashStr_regexp, replace the placeholder text with a regular expression, conformant with the syntax restrictions given at the top of this question, describing the set of strings as given above.
  • In the file abdashStr_rrg, replace the placeholder text with a (right) regular grammar describing the same set of strings.
